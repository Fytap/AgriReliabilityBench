# Data and code availability

Repository: https://github.com/Fytap/AgriReliabilityBench

Release: `1.0.0`

The public repository contains code, configuration files, derived result tables, figure assets, supplementary materials, audit reports, and reproducibility scripts for the benchmark.

Raw CropAndWeed, PhenoBench, CWFID, and CWD30 data are not redistributed and should be obtained from the original dataset providers under their respective licenses. The repository provides dataset configuration files, manifest-generation scripts, label-harmonization rules, evaluation scripts, smoke-test utilities, and derived aggregate result records for reproducing and auditing the benchmark after local dataset installation.

If a Zenodo or OSF archive is created for the final journal submission, add the version-specific DOI here and update `CITATION.cff` and `.zenodo.json`.
